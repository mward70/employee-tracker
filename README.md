## Description
This application allows for the tracking of employees including the updating of their job titiles, salaries, etc.Employees and certain demographic information can be entered, updated, and deleted.

## Installation Instructions
Open the terminal. Run 'npm i' to install any needed packages. Run 'npm start'.
